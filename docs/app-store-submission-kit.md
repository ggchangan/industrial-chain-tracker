## 一句话简介

候选：

1. 用图谱、原文和逻辑卡片理解产业链变化。
2. 面向产业研究的结构化阅读与跟踪工具。
3. 把产业链研究、逻辑变化和市场验证放在一个地方。

推荐首版使用：

```text
用图谱、原文和逻辑卡片理解产业链变化。
```

## 应用简介草稿

```text
产业链研究库是一款面向产业研究和行业观察的结构化阅读工具。

你可以在这里查看不同产业链的结构图谱、核心逻辑、最新研究、跟踪验证和完整研究原文。应用将分散的行业资料、公司公告、自有研究和逻辑卡片组织在同一条产业链下，帮助你快速理解一条产业链正在发生什么、关键变量是什么、后续应该跟踪哪些变化。

首版能力包括：
- 产业链列表和跨产业链搜索
- 产业链详情与图谱预览
- 当前重点、最新研究和核心逻辑
- 跟踪验证和观察项
- 完整研究原文按需阅读
- 个股 K 线入口，用于观察市场是否交易相关产业逻辑

重要提示：本应用提供行业研究资料整理和阅读辅助，不构成证券投资建议。应用中的产业逻辑、公司信息、K 线外链和跟踪验证仅供研究参考，用户应自行判断风险。
```

## 截图清单

建议至少准备 4 张核心截图：

1. 首页产业链列表与搜索  
   说明文案：`快速浏览和搜索重点产业链`
2. 产业链详情当前重点  
   说明文案：`先看当前变化，再深入研究逻辑`
3. 核心逻辑 / 最新研究  
   说明文案：`把资料沉淀为可跟踪的产业逻辑`
4. 原文阅读 / 个股集中营  
   说明文案：`阅读完整原文，并用 K 线观察市场验证`

iOS 需要按 App Store Connect 要求提供对应设备尺寸截图。Android 各市场通常需要手机截图，部分市场还要求应用图标、宣传图或隐私截图。

## 审核说明

可在审核备注中使用：

```text
本应用为产业链研究资料阅读工具。首版无需登录即可浏览公开内容，包括产业链列表、搜索、详情页、图谱、研究原文和市场验证入口。

应用不提供证券交易功能，不提供投资顾问服务，也不承诺投资收益。个股 K 线入口仅跳转至公开财经页面，用于辅助用户观察市场走势。

首版 iOS/Android App 暂未开放账号登录、推送和离线缓存。收藏、订阅和跨端阅读历史将在后续账号体系完善后开放。
```

如果审核人员需要测试路径：

```text
打开 App -> 首页选择任一产业链 -> 查看当前重点 / 核心逻辑 / 最新研究 -> 点击“加载完整原文”阅读正式文章。
```

## 隐私问卷口径

首版阅读版实际行为：

- 不读取通讯录。
- 不读取相册。
- 不使用相机。
- 不采集定位。
- 不读取手机状态。
- 不做跨 App 跟踪。
- 不提供第三方广告。
- 通过 HTTPS 请求公开内容 API 和静态资源。
- 可在本地保存阅读进度等设备内状态。
- 微信小程序可使用微信登录；iOS/Android 首版 App 暂不开放账号登录。

应用市场问卷填写时，应以实际构建版本为准。如果后续打开 Apple 登录、手机号登录、推送或统计 SDK，需要同步更新隐私政策和问卷。

## 权限说明

Android 首版权限：

```text
INTERNET：访问产业链内容 API 和静态资源。
ACCESS_NETWORK_STATE：判断网络状态，便于展示加载失败提示。
VIBRATE：用于轻量交互反馈，首版可保留，也可按市场要求移除。
```

iOS 首版不申请相机、定位、通讯录、相册等敏感权限。
